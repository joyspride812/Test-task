Тестовое задание

Все действия выполнять на ОС РЕД ОС
Ссылка на образ ОС: https://redos.red-soft.ru/product/downloads/
Необходимая документация есть в Базе Знаний Ред ОС. https://redos.red-soft.ru/base/

1. развернуть домен на основе Samba DC (есть в репозитории) с двумя контроллерами домена и репликацией. Разворачивать с вариантом DNS  BIND_DLZ

- Развернуть домен
- Добавить второй контроллер домена
- Проверить репликацию

2. Развернуть на одном из контроллеров сервер управления РЕД АДМ Стандартная редакция (есть в репозитории)

3. развернуть клиентскую ОС 
Ввести в домен, продемонстрировать доступ к интерфейсу управления РЕД АДМ.

Playbook "dc_install.yml" устанавливает сервер SAMBA DC.
Необходимо указать значения переменных в файле .\group_vars\redsoft-dc\vars.yaml :
domain - имя разворачиваемого домена
pdc - имя главного домен контроллера
pdc_ip: ip главного домен контроллера

Если имя хоста из файла inventory.yml не совпадает со значением переменной pdc, сервер SAMBA DC устанавливается, как  дополнительный.
